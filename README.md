# Enterprise CRM Backend

An enterprise CRM backend built with .NET 8 and Clean Architecture.

## Features

- JWT Authentication
- Clean Architecture
- CQRS (MediatR)
- Entity Framework Core
- SQL Server
- Repository Pattern
- Global Exception Handling
- Swagger API Documentation

## Project Structure

- CRM.api
- Crm.Application
- Crm.Domain
- Crm.Infrastructure
